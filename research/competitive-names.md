# Competitive Research: AI Agent Trust/Scoring Products
**Research Date:** 2026-03-03

---

## Executive Summary

The AI agent trust/reputation space is rapidly emerging with significant competition from established players and new entrants. Key findings include:

- **Solana's official Agent Registry** (solana.com/agent-registry) is the most direct competitor using ERC-8004 standard
- **Multiple "Sentinel"** branded products exist in blockchain/AI security
- **"Ghost"** naming conflicts found with GhostSpeak (Solana agent commerce protocol)
- **Privado ID + SingularityNET** partnership creating the "AI Agent Trust Registry"
- **ERC-8004** is emerging as the dominant standard for agent identity/reputation

## 1. Direct Competitors - Agent Trust/Reputation Systems

### 1.1 Solana Agent Registry / SATI (Solana Agent Trust Infrastructure)
- **URL:** https://solana.com/agent-registry
- **What they do:** Official Solana implementation of ERC-8004 standard for agent identity and reputation
- **Features:**
  - Live dashboard for registered agents
  - Reputation scores and attestations
  - Explorer for 8004 ecosystem
  - Marketplace for discovering agents with on-chain identity/reputation scores
- **Stage:** Live/production
- **Similarity:** VERY HIGH - Direct competitor for agent trust scoring
- **Trademark Issues:** Low - "Agent Trust Infrastructure" is descriptive

### 1.2 Trusta.AI (Trust Labs)
- **URL:** https://www.trustalabs.ai/
- **CoinMarketCap:** https://coinmarketcap.com/currencies/trusta-ai/
- **What they do:** 
  - TrustScan: Sybil detection using behavior fingerprinting
  - TrustGo: Wallet intelligence and credit scoring (MEDIA score)
  - AI Agent Network with modular trust layers
- **Funding:** Has token (TA), actively traded
- **Stage:** Production
- **Similarity:** HIGH - Direct competitor in wallet/agent reputation scoring
- **Trademark Issues:** Medium - "Trusta" is brandable, but "Trust" is generic

### 1.3 GhostSpeak (Solana AI Agent Commerce Protocol)
- **URL:** https://github.com/Ghostspeak/GhostSpeak
- **What they do:** 
  - AI Agent Commerce Protocol on Solana
  - "Ghost Score" - Multi-source reputation system with 8+ data inputs
  - Credential issuance with W3C VCs
  - Crossmint EVM bridging
- **Stage:** Active development (multisig planned for mainnet)
- **Similarity:** HIGH - Direct competitor with "Ghost" naming
- **Trademark Issues:** HIGH - "Ghost" + agent scoring = potential confusion with any "Ghost" branded product

### 1.4 Privado ID + SingularityNET Partnership
- **URL:** https://www.privado.id/blog/singularitynet-x-privado-id-making-ai-trustworthy
- **Announced:** March 4, 2025
- **What they do:**
  - "AI Agent Trust Registry" (exact name!)
  - Decentralized identifiers (DIDs) for AI agents
  - Verifiable credentials via zero-knowledge proofs
  - Communication layer for verifying agent safety/fairness
- **Funding:** Well-funded (SingularityNET is established)
- **Stage:** Pilot phase (AIRIS AI Agent integration)
- **Similarity:** VERY HIGH - Using "AI Agent Trust Registry" name
- **Trademark Issues:** VERY HIGH - Direct name conflict with "AI Agent Trust Registry"

### 1.5 Reclaim Protocol
- **URL:** https://reclaimprotocol.org/
- **Docs:** https://docs.reclaimprotocol.org/

### 1.6 SATI (Solana Agent Trust Infrastructure) - separate implementation

## 9. Recommendations

### 9.1 Naming Strategy
1. **AVOID:** "AI Agent Trust Registry" - Direct conflict with Privado ID/SingularityNET
2. **AVOID:** "Ghost" + agent scoring - Conflict with GhostSpeak's "Ghost Score"
3. **CAUTION:** "Sentinel" - Crowded space, consider alternatives
4. **CONSIDER:** Unique coined names (e.g., "Spectral" model)
5. **CONSIDER:** Descriptive but distinctive combinations

### 9.2 Positioning Opportunities
1. **Solana-native focus** - Differentiate from Ethereum-first competitors
2. **Developer tooling** - API-first approach vs. consumer-facing
3. **Specific verticals** - DeFi, NFTs, gaming reputation
4. **Privacy-preserving** - ZK-based reputation (less common)
5. **Composability** - Easy integration with existing protocols

### 9.3 Key Differentiators to Consider
- **Speed:** Sub-second finality (Solana advantage)
- **Cost:** Low fees for reputation updates
- **Developer Experience:** Simple SDK vs. complex standards
- **Multi-chain:** Cross-chain reputation portability
- **Privacy:** Optional anonymous reputation
